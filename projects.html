ElderVR — Adaptive VR Healthcare System

ElderVR is designed to address accessibility and cognitive load for elderly populations. The project integrates low-cognitive visual affordances, deliberate interaction feedback, and caregiver-safe fallbacks. It is the result of multi-round field testing with demographics-focused sampling, and is described in a reproducible pipeline below.

DDAP — Detecting Digital Addiction Patterns
